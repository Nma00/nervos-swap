https://youtu.be/G8zOd8IV0E0

## Build

### Backend

Install dependencies
```
npm install
```
Compile contracts
```
npx hardhat compile
```

Deploy contracts
```
npx hardhat run .\scripts\deploy-script.js
```

### Sdk

Install dependencies
```
npm install
```
```
npm run build
```

### Interface

Install dependencies
```
npm install
```

Start development server
```
npm start
```

Production build
```
npm run build
```
