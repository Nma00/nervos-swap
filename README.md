# Nervos Swap
Uniswap fork for Nervos L2 Godwoken.

https://youtu.be/O501_cvzmm4

## Build

### Backend

Install dependencies :
```
npm install
```
Compile contracts :
```
npx hardhat compile
```

Deploy contracts :
```
npx hardhat run .\scripts\deploy-script.js
```

### Sdk

Install dependencies and build :
```
npm install
```
```
npm run build
```

### Interface

Install dependencies :
```
npm install
```

Start development server :
```
npm start
```

Production build :
```
npm run build
```
